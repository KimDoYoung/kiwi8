# 코딩 지침

## 3개 증권사의 표현

- 3개의 증권사는 순서를 갖는다.
    - 1번째는 한국투자증권, 2번째는 LS증권으로,  3번째는 키움증권으로
- 3개의 증권사는 색상을 갖는다.
    - 한국투자증권 : #80624c, LS 증권 : #003378, 키움증권 : #e4007f
- 3개 회사의 약어는  KIS, LS, Kiwoom 이다.
- 3개 회사의 한글 약어는 한투, 엘에스, 키움 이다.
-  위 순서와 색상을  준용한다.

## 원칙

- typescript를 사용한다. strict mode로 컴파일한다.
- shared 폴더의 공통 컴포넌트를 적극적으로 사용한다.
- shared 폴더내의 컴포넌트는  다른 페이지 폴더에서 재사용이 가능하게 일반화 하여야 한다.
- format함수는  shared/lib/utils.ts 에서 찾아서 사용한다. Page내에서 새로 만들지는 않는다.


## 공통 component

- shared/components/ 에 공통 컴포넌트를 정의한다.
- 용도와 사용법을 기술한다.

## 로딩 표시

- 기본적으로 다음과 같이  로딩을 구현한다.

```
    if (isLoading) {
        return <Loading message="계좌 정보를 불러오는 중..." />
    }

    if (isError || !data) {
        return <LoadingFail message="계좌 정보를 불러오는데 실패했습니다." />
    }
```
