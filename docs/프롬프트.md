## 프롬프트

### 마켓 선택 

1. 현재 한국은 2개의 시장 KRX와 NXT 2개가 있음.
2. 주식의 현재가를 구하기 위해서는 지금 현재 시각을 기준으로 2개의 시장 중 1개를 선택해야함.
3. 장 오픈 데이 즉 장이 열리는 날인지 판단.
4. 열리는 날이라면 시각이 09:00-15:30 사이 인지 판단. 그 시각이라면 KRX
5. 나머지는 NXT에서 현재가를 가져와야함.
```
즉 
market_choice_for_price() # 가격정보를 가져오기 위해서 시각을 기준으로 KRX,NXT 선택
market_choice_for_trade() # 매매를 위해서 시각을 기준으로 KRX,NXT, NULL인 경우 매매 시간이 아님
```

6. 현재날짜(현재시각)이 장이 오픈날인지는 판단하는 로직
    - cache_manager에서 찾아본다. get('OPENDAY','yyyymmdd')  'Y' 인 경우 현재 시각을 기준으로 선택한다.
    - 'N' 인경우 NXT
    - 값이 없는 경우
        - 토,일요일이면 'N' 으로 처리, cache_manger에 추가한다.
        - 
7. OpenTimeChecker에 2개의 함수 를 넣으면 어떨까?    

#### NXT

2. main market : 09:00:30 - 15:20
3. after market : 15:40 - 20:00
